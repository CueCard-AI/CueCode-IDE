# TOML

TOML support is available through the [TOML extension](https://cuecode.dev/extensions/toml).

- Tree-sitter: [tree-sitter/tree-sitter-toml](https://github.com/tree-sitter/tree-sitter-toml)

A TOML language server is available in the [Tombi extension](https://cuecode.dev/extensions/tombi).
